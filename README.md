# 💰 Smart Expense Tracker

A Smart Expense Tracker built using **Python, Streamlit, Pandas, Matplotlib, and SQLite**. This application helps users manage their daily expenses, monitor budgets, analyze spending patterns, and generate insightful reports through an interactive dashboard.

---

# 📌 Features

## 🔐 User Authentication
- User Registration
- User Login
- Secure Authentication using SQLite
- Logout

## 💵 Expense Management
- Add New Expense
- Edit Existing Expense
- Delete Individual Expense
- Delete All Expenses

## 📊 Dashboard
- Total Spending
- Average Expense
- Highest Spending Category
- Recent Transactions

## 💰 Budget Management
- Set Monthly Budget
- Budget Progress Bar
- Remaining Budget
- Budget Exceeded Alert

## 📈 Reports & Analytics
- Category-wise Pie Chart
- Category-wise Bar Chart
- Expense Trend Graph
- Top 5 Highest Expenses
- Monthly Summary
- Smart Spending Suggestions

## 📋 Records
- View All Expenses
- Search by Description
- Filter by Category
- Download Expenses as CSV

# ⚙ Installation

## Step 1: Clone the Repository

```bash
git clone https://github.com/balajinaveen2006-w/Smart-Expense-Tracker.git
```

## Step 2: Open the Project Folder

```bash
cd smart-expense-tracker
```

## Step 3: Create a Virtual Environment (Optional)

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

## Step 4: Install Dependencies

```bash
pip install -r requirements.txt
```

## Step 5: Run the Application

```bash
streamlit run app.py
```

The application will open in your browser at:

```
http://localhost:8501
```
